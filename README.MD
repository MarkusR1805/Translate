# 🌐 Helsinki-NLP Übersetzer

Ein lokaler Offline-Übersetzer mit grafischer Benutzeroberfläche (GUI) basierend auf Helsinki-NLP Modellen.

![Screenshot](screenshots/app-screenshot.png)

## ✨ Features

- **Offline-Übersetzung** – Keine API-Keys, keine Cloud, keine Kosten
- **Mehrsprachig** – Unterstützt:
  - Englisch ⇄ Deutsch
  - Französisch ⇄ Deutsch
- **Dark Theme** – Angenehme, augenschonende Oberfläche
- **Einfache Bedienung** – Copy & Paste aus der Zwischenablage
- **Automatischer Download** – Modelle werden beim ersten Start automatisch geladen
- **macOS optimiert** – Native Look & Feel

## 📦 Installation

### Voraussetzungen

- Python 3.9 oder höher
- macOS, Linux oder Windows

### Schritte

1. **Repository klonen**
   ```bash
   git clone https://github.com/DEIN_USERNAME/helsinki-translator.git
   ```
